# Collection field: Handwritten Signature

This plugin adds a handwritten signature field for collections, suitable for scenarios such as e-sign receipt confirmation and approval signing.

When configuring the field, you must explicitly choose a **file collection** to store signature images. This field depends on **File manager** to provide file collections, uploads, and storage capabilities. The actual storage behavior is determined by the target file collection's own configuration.

In UI configuration, you can also adjust signature-related options such as pen color, background color, canvas size, and thumbnail display size.
