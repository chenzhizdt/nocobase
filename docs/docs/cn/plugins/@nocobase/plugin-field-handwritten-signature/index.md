# 数据表字段：手写签名

该插件为数据表提供“手写签名”字段类型，适用于电子签收、审批签名等场景。

配置字段时，需要显式选择一个 **文件数据表** 来保存签名图片。该字段依赖 **文件管理器** 提供文件数据表、上传与存储能力；实际使用的存储配置由目标文件数据表决定。

在界面配置中，还可以调整签名相关参数，例如笔触颜色、背景颜色、签名画布尺寸，以及缩略图显示尺寸。
